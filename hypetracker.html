Hype Tracker

A full stack web application that visualizes the popularity of sneakers.

Overview

This project was part of a one month final project for an internet computing course. My primary responsibilities for this project included designing the database, and programming the back-end.

Problem

The idea of this project began with our curiosity of whether a specific sneaker was going to sell out or not. One way to track this is to visit different social media websites, such as Twitter, Instagram, and Reddit, and calculate the number of comments and likes, posts about that specific sneaker accumulates. The more popular a sneaker is, the more likely it is to sell out.

Database

I first set out to sketch an Entity-Relation diagram of a database that would contain unreleased sneakers. Each sneaker would have a ‘Ranking’. The rankings would be used to calculate a score from different social media platforms based on the number of mentions/likes the sneaker’s name accumulated.

In addition to just viewing sneakers popularity on social media platforms, we also enabled users to register for an account in order to add sneakers to their ‘Watchlist’. The Watchlist feature allows users to easily track the popularity of their favourite sneakers. After finalizing the ER diagram, I began implementing it in MySQL.

Final Website

I utilized Twitter and Reddit API’s to search the number of times the sneaker’s name was mentioned on the platform. Once per day, a request was made to the API’s and the calculated scores were stored in the database. The Twitter score is shown by text, while the Reddit score is plotted on a line chart, and then displayed on that specific sneaker's details page.

Reflection

Because of strict-time frames, more emphasis was put on implementing the core features of the back-end of this project than the front-end. This project has helped me further develop my programming skills in languages such as PHP, HTML, CSS, and JavaScript. In addition, I gained a deeper understanding on how to extract data from other websites using their APIs.
